# 任务与链接管理器 — 设计规格

**日期:** 2026-06-29  
**状态:** 已批准，进入实现

## 1. 产品目标

个人任务与链接管理 Web 应用，核心工作流：**收集箱 → 本周任务 → 今日待办**。

## 2. 需求定稿

| 维度 | 决策 |
|------|------|
| 使用者 | v1 单人无登录；schema 预留 `user_id` |
| 条目类型 | 双模式：任务 / 收藏 |
| 收藏限制 | 收藏只能留在收集箱 |
| 流转规则 | 主流程单向，允许回退一层 |
| 完成行为 | 移入已完成归档区，可恢复/删除 |
| 时间边界 | 智能滚存；周一起算 |
| 字段 | 标题、链接、备注、标签（预设+自定义） |
| 链接优先级 | 国内优先：微信/飞书/小红书/B站，YouTube 次之 |
| 快速收集 | 顶部输入框 + Bookmarklet |
| 搜索 | 全局搜索，含归档，标签筛选 |
| 界面 | 桌面三栏 + 移动 Tab |
| 语言 | 中英双语 |

## 3. Stage 迁移规则

- bookmark 仅 inbox，禁止迁出
- task: inbox→week→today 允许；today→week、week→inbox 回退允许
- 完成 → archived；恢复 → inbox

## 4. 智能滚存

- 每日：today 未完成保持 today，提示延续项数量
- 每周一：week 未完成保留 week，更新 updated_at
